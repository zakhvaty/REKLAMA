# REKLAMA Advertising Platform

Welcome to REKLAMA, a modern advertising platform designed to deliver targeted advertisements to your audience.

## Features

- RESTful API for advertisement management
- Responsive web interface
- Easy integration with existing systems
- Scalable architecture

## Project Structure

```
REKLAMA/
├── src/              # Backend source code
├── public/           # Frontend static files
├── assets/           # Media assets
├── components/       # Reusable UI components
├── services/         # Business logic and services
├── utils/            # Utility functions
├── styles/           # CSS and styling files
└── tests/            # Test files
```

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd REKLAMA
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

To start the development server:
```bash
npm run dev
```

To start the production server:
```bash
npm start
```

The application will be available at `http://localhost:3000`

## API Endpoints

- `GET /` - Home page
- `GET /api/ads` - Get all advertisements
